Intruções Gerais e tipos e arquivos

O curso está ligado a um repositório na plataforma virtual Github, que deve ser acessada pelo link (https://github.com/raquelribeiro/Introducao-ao-ambiente-R)

Neste ambiente, vocês encontrarão vários arquivos que servirão como material de apoio ao longo do curso. Os arquivos estarão em diferentes formatos (PDF, md, Rmd, dentre outros). É importante que os alunos se familiarizem com a plataforma GitHub e, para isso, disponibilizamos alguns links úteis:

  • https://tableless.com.br/tudo-que-voce-queria-saber-sobre-git-e-github-mas-tinha-vergonha-de-perguntar/
  • https://pt.wikipedia.org/wiki/GitHub
  • https://www.youtube.com/watch?v=vJjGSxvh7TY

Um dos formatos mais importantes que utilizaremos é o Rmd, um formato de arquivo que permite que o usuário do RStudio, importe e execute os comandos contidos no arquivo, diretamente na plataforma

Após realizar as leituras preliminares, tente importar para o seu RStudio o arquivo do repositório denominado "exemplo.Rmd"
